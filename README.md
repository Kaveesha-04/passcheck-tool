# PassCheck: Secure Password Utility

## 🌍 The Problem
In the modern digital landscape, weak passwords are the leading cause of personal data breaches. Many users struggle to create complex passwords or understand the vulnerability of their current credentials.

## 🛠️ The Solution
**PassCheck** is a lightweight, responsive web application designed to help users:
1. **Analyze:** Get real-time feedback on password entropy and strength.
2. **Generate:** Create randomized, high-strength passwords instantly.
3. **Educate:** Learn best practices for digital identity protection.

## 🚀 Built With
* **HTML5** - Semantic structure.
* **CSS3** - Modern, responsive design (Cybersecurity Dark Theme).
* **JavaScript (Vanilla)** - Real-time strength evaluation logic.
